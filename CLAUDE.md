# Libe Libe Jewel — Telegram Mini App

Интернет-магазин украшений ручной работы с натуральными камнями.
Telegram: t.me/libe_libe_acc
Слоган: "made with love"
Аудитория: женщины, интересующиеся астрологией, нумерологией, таро.

## Технический стек

- Telegram Mini App (TWA)
- HTML + CSS + JavaScript (без фреймворков)
- Товары: Google Sheets → публичный CSV URL
- Заказы: Telegram Bot API (токен и chat_id подставим позже)

## Страницы приложения

1. **Главная** — баннер, слоган, категории, новинки
2. **Каталог** — сетка товаров, фильтр по категориям
3. **Карточка товара** — фото, название, камень, цена, описание, кнопка в корзину
4. **Корзина** — список товаров, итого, кнопка заказать
5. **Оформление заказа** — имя, телефон, доставка, комментарий
6. **Успешный заказ** — страница подтверждения

## Google Sheets — структура таблицы

Столбцы: `id`, `название`, `категория`, `цена`, `описание`, `камень`, `фото_ссылка`, `в_наличии`

Подключение через публичный CSV URL:
```
https://docs.google.com/spreadsheets/d/SHEET_ID/export?format=csv
```
(SHEET_ID подставим когда создадим таблицу)

---

## Telegram Bot

- Токен бота: `ПОДСТАВИТЬ_ПОЗЖЕ`
- Chat ID владельца: `ПОДСТАВИТЬ_ПОЗЖЕ`
- При заказе бот отправляет сообщение владельцу с деталями заказа

## Правила разработки

- Стек максимально простой — новичок должен понимать каждую строку
- Не усложнять без необходимости
- Объяснять каждый шаг простыми словами
- Делать по этапам, ждать подтверждения перед следующим шагом
